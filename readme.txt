== Description =
### Check System Details - Easily Check Your Installation and Server Details ###

##No Setup Required##

## The details include:  ##

#WordPress Details#
* WordPress address
* Site address
* WordPress version
* WordPress multisite
* WordPress memory
* WordPress debug
* WordPress debug
* WordPress cron
* WordPress Language

#Server Details#
* Max input timeout
* Max input vars
* Max file upload
* Max file upload

#Database Details#
* Database Name
* Database Host
* Database Charset
* Database Prefix

#Names of existing Database Tables#

#Current Theme Details#
* Name
* Version
* Description
* Author

#Installed Plugin Details#
* Plugin name with hyperlink to plugin's website
* Version
* Author
